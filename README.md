# Sistema-Gerenciamento-Tarefas

### Especificação técnica 
Projeto de gerenciamento de tarefas do tipo CRUD usando linguagem de programação JAVA , 
utilizo conceitos de Programação Orientada Objetos (POO) e de arquitetura MVC (Model , View , Controller ).
Além de manipulação de arquivos do tipo JSON.
