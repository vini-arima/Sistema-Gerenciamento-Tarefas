# Sistema-Gerenciamento-Tarefas

## Descrição Técnica 

Projeto feito em linguagem JAVA, Nesse projeto utilizo os conceitos de arquitetura de software \
escolhi a arquitetura MVC ( Model , View , Controller ) para estrutura as pastas. 

Para base de dados utilizo o JSON e utilizo o padrão CRUD para gerenciar as tarefas.

### O que aprendi 
- Aprendi manipulação de dados em JSON usando JAVA 
- Aprendi a usar biblioteca externa lib sem usar Maven ou Grandle
- Revisei conceito de CRUD

### Como executar o código ?
No Vscode:

Executar ou Depurar --> Run Java (No arquivo Main.java)

### Possivel problema ao rodar o código
Para executar o programa você precisar utilizar \
a biblioteca externa *json-20231013.jar*. Caso 
não tenha o programa dara erro.
