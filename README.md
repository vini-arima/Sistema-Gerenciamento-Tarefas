# Sistema-Gerenciamento-Tarefas

## Descrição Técnica 

Projeto feito em linguagem JAVA, Nesse projeto utilizo os conceitos de arquitetura de software \
escolhi a arquitetura MVC ( Model , View , Controller ) para estrutura as pastas. 

Para base de dados utilizo o JSON e utilizo o padrão CRUD para gerenciar as tarefas.
